# Beast

A heavy display typeface from fonderie.download. Single weight.

- **License:** WTFPL (do whatever you want) ([`LICENSE.txt`](./LICENSE.txt))
- **Upstream:** https://fonderie.download/beast.html
- **Format:** `woff2`

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/maxlair1/open-typefaces@v1.5.0/fonts/beast/beast.css">
```

```css
body { font-family: 'Beast', sans-serif; }
```
