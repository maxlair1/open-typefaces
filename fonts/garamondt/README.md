# Garamondt

A free Garamond-inspired serif (regular + italic) from fonderie.download.

- **License:** WTFPL (do whatever you want) ([`LICENSE.txt`](./LICENSE.txt))
- **Upstream:** https://fonderie.download/garamondt.html
- **Format:** `woff2`

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/maxlair1/open-typefaces@v1.5.0/fonts/garamondt/garamondt.css">
```

```css
body { font-family: 'Garamondt', serif; }
```
