# Isenheim

A gothic / textura-inspired display face by Benoît Ferran, via [Tunera Type Foundry](https://www.tunera.xyz/fonts/isenheim/).

- **License:** SIL Open Font License 1.1 ([`OFL.txt`](./OFL.txt))
- **Upstream:** https://www.tunera.xyz/fonts/isenheim/ · https://gitlab.com/benoitsansterre/isenheim
- **Weights:** Régulier (400), Fin (200). Display use.
- **Format:** `woff2` (converted from OTF).

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/maxlair1/open-typefaces@v1.6.0/fonts/isenheim/isenheim.css">
```

```css
h1 { font-family: 'Isenheim', serif; }
.light { font-weight: 200; }
```
