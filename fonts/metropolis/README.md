# Metropolis

A modern, geometric sans-serif by Chris Simpson, released into the **public domain**.

- **License:** The Unlicense (public domain) ([`LICENSE.txt`](./LICENSE.txt)) — no restrictions at all.
- **Upstream:** https://github.com/dw5/Metropolis — a preserved fork; the original `chrismsimpson/Metropolis` repo was taken down in 2020.
- **Weights:** 100–900 (Thin · ExtraLight · Light · Regular · Medium · SemiBold · Bold · ExtraBold · Black), each with an italic.
- **Format:** `woff2` only (converted here from the upstream TTF — upstream shipped only `woff`/`ttf`/`otf`).

## Use it

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/maxlair1/open-typefaces@v1.4.0/fonts/metropolis/metropolis.css">
```

```css
body { font-family: 'Metropolis', system-ui, sans-serif; }
strong { font-weight: 700; }
```

> Pin to a **tag** (`@v1.4.0`) or **commit SHA** in production.
