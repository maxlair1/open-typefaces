# Mioya

A display face by Sora Sagano (dotcolon), with a variable weight axis (100–1000).

- **License:** SIL Open Font License 1.1 ([`OFL.txt`](./OFL.txt))
- **Upstream:** https://dotcolon.net/fonts/mioya
- **Format:** `woff2`

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/maxlair1/open-typefaces@v1.5.0/fonts/mioya/mioya.css">
```

```css
body { font-family: 'Mioya', sans-serif; }
```
