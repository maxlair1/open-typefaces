# Optician Sans

A display typeface by [ANTI Hamar & Fábio Duarte Martins](https://optician-sans.com/), based on the historical eye charts and optotypes used by opticians worldwide.

- **License:** SIL Open Font License 1.1 ([`OFL.txt`](./OFL.txt)) — free to use, embed, and redistribute.
- **Upstream:** https://github.com/anewtypeofinterference/Optician-Sans (not on Fontsource / Google Fonts).
- **Weight:** single weight (Regular / 400). No italics, not variable.
- **Use for:** large headings, logos, numerals — it's a display face, not a body font.
- **Format:** `woff2` only.

## Use it

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/maxlair1/open-typefaces@v1.3.0/fonts/optician-sans/optician-sans.css">
```

```css
h1 { font-family: 'Optician Sans', sans-serif; }
```

> Pin to a **tag** (`@v1.3.0`) or **commit SHA** in production. Since there's only one weight, asking for `font-weight: 700` will trigger faux-bold synthesis — stick to `400`.
