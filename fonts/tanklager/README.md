# Tanklager

A heavy industrial display face by Ariel Martín Pérez, via [Tunera Type Foundry](https://www.tunera.xyz/fonts/tanklager/).

- **License:** SIL Open Font License 1.1 ([`OFL.txt`](./OFL.txt))
- **Upstream:** https://www.tunera.xyz/fonts/tanklager/ · https://codeberg.org/ArielMartinPerez/Tanklager
- **Cuts:** Original (normal width) and Kompakt (compact). Variable `wdth` axis 80–100.
- **Format:** `woff2`.

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/maxlair1/open-typefaces@v1.6.0/fonts/tanklager/tanklager.css">
```

```css
h1 { font-family: 'Tanklager', sans-serif; }            /* Original */
.compact { font-family: 'Tanklager'; font-stretch: 80%; } /* Kompakt */
```

Use `tanklager-variable.css` for the smooth `wdth` axis (set any `font-stretch` 80%–100%).
